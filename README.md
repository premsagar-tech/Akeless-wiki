## Overview

This repository contains a modular book and labs to learn, deploy, and operate an enterprise-grade secrets management solution on AKS using **Akeyless** and the **External Secrets Operator (ESO)**.

- Chapters 1–8 (theory)
- Labs for each chapter (hands-on)
- Troubleshooting, runbooks, GitOps & production hardening
